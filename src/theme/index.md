## themes

| theme             |         | navigation                         |
| ----------------- | ------- | ---------------------------------- |
| alabaster         | default | toplevelのtoctree 固定                |
| basic系            | builtin | prev, next, table of contents, 現在地 |
| rtd               |         | toplevelのtoctree + 現在地, prev, next |
| sphinx_book_theme |         | toplevelのtoctree + 現在地, prev, next |

-   rtd の `toplevelのtoctree + 現在地` が使い易い
