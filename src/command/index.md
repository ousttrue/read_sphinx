# Command

## builtin
### sphinx-quickstart

ひな型になる conf.py の生成など。

### sphinx-build

html の生成など

### sphinx-apidoc

`python` のソースからAPIドキュメントになる `rst` を生成する。

* `-F`: conf.py, index.rst なども生成する

## 外部コマンド

### sphinx-autobuild

`pip install sphinx-autobuild`

preview server。ファイルの更新を監視して自動ビルドする機能と、HTML のホスティング。
